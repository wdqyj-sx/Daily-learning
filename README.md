  ### Promise基本实现（2021-9-4）
    + Pormise是一个类，无需考虑兼容性
    + 使用Promise会传入一个执行器(executor)，执行器是立即执行
    + 当前executor给了两个函数来操作当前状态，Promise有三个状态，成功态，失败态，等待态，默认等待态，执行resolve，进入成功态，执行reject进入失败态
    + 每个Promise实例都有一个then方法
    + Promise状态改变，不能更改
